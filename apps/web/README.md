# Maliyet Web

React + TypeScript + Vite tabanlı web frontend başlangıcıdır. Masaüstü uygulamayı değiştirmez; API üzerinden aynı `urun_maliyet_db` veritabanına ulaşacak şekilde geliştirilecektir.

Yerel geliştirme komutu:

```powershell
cd apps/web
npm install
npm run dev
```

Varsayılan API adresi:

```text
http://127.0.0.1:8100
```
